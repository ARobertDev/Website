This is the Page for the Online Campaign that runs the Tango RPG System, using a custom campaign called The Hunt for Power. If you're looking for the In-Person DnD Campaign hosted by the Palmer High School Tabletop RPG Club, please look at the page labelled "IN-PERSON."

Welcome to the Online Campaign! This Campaign's Game Master (GM) is Andrew Martin, the owner, operator, and designer of this website. Andrew is also the creator of The Hunt for Power Campaign which was created using the free Tango RPG System. This campaign currently has no set date to begin, and no set schedule. When this changes this channel will be notified. Even after starting, we will still be accepting new characters, so please feel free to join anyways.
